Violations

Random Death Match (RDM)

  1. Introduce yourself professionally.
  2. Gather usernames, summary, location, and evidence.
  3. Use view and server logs to verify claims.
  4. Speak to both parties.
  5. Action examples:
    • 1st Offense: Warning
    • 2nd Offense: Kick
    • Mass RDM: Permanent ban
  6. Log with:Melonly

Fail Roleplay (FRP)

  1. Confirm if player broke roleplay logic (e.g., unrealistic actions).
  2. Educate on proper RP if it's a first offense.
  3. Repeated violations:
    • 1st Offense: Warning
    • 2nd Offense: Kick
    • Permanent ban (depending on severity)
  4. Log with:Melonly

Combat logging/Left to avoid punishment (LTAP)

  1. Use logs to check if the player left mid-scene.
  2. If confirmed, issue:
    • warn or kick for first offense
    • ban for repeat offenses
  3. Log with:Melonly

No Intrest To RolePlay (NITRP)

  1. If a player refuses to RP seriously:
    • Give a verbal warning.
    • If continued: kick or ban.
  2. Log with:Melonly

Staff Disrespect

  1. Remain calm and do not retaliate.
  2. Warn for first offense.
  3. Kick or ban if it continues.
  4. Escalate to Higher Ups if necessary.
